.Modelica.Math.Special.Internal.polyEval

Information

Evaluate a polynomial using Horner's scheme.

Interface

function polyEval
  input Real c[:] "Polynomial coefficients";
  input Real u "Abscissa value";
  output Real y "= c[1] + u*(c[2] + u*(c[3] + u*(c[4]*u^3 + ...)))";
end polyEval;
