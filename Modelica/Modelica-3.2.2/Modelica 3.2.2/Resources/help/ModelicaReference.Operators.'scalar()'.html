.ModelicaReference.Operators.'scalar()'

Information

Returns a one-element array as scalar

Syntax

scalar(A)

Description

Returns the single element of array A. size(A,i) = 1 is required for 1 ≤ i ≤ ndims(A).

Examples

Real A[1,1,1] = {{{3}}};
Real e = scalar(A);  // = 3
